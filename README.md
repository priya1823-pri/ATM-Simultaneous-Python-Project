# ATM-Simultaneous-Python-Project

--> ATM Simulator written in Python
--> Console-based system (runs in terminal)

--> Features included:
      -Account statement
      -Withdraw money
      -Deposit money
      -Change PIN
      
-->  User must enter a valid username and PIN to log in
-->  After login, all features become available
-->  Deposit/withdrawal updates the account balance instantly
-->  All transactions can be viewed in the account statement
-->  PIN can be changed by entering a new PIN and confirming it
-->  No database or external files used
-->  All user data (PIN, balance) is stored inside the source code
